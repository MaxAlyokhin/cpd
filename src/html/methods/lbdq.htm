МЕТОДИКА:                                  
             Лидерский и гендерный стиль поведения менеджера                
                          (А. Кэнн, Д. Зигфрид)                             
                         (A. Cann, D. Siegfried)                            
                       (модификация Т.В. Бендас).                           
 
      Методика предназначена для изучения лидерского  и  гендерного  стилей 
 поведения менеджера.                                                       
 
      Гендерный стиль - стиль поведения индивида, соответствующий гендерным 
 стереотипам для данного пола.                                              
 
      Половые особенности лидерского стиля.                                 
      Наиболее распространенными считаются 6 стилей поведения лидера, кото- 
 рые отражают разные аспекты этого поведения и не  обязательно  сводятся  к 
 дихотомии "инструментальный - экспрессивный":                              
      ориентированный на задачу, или деловой;                               
      ориентированный на взаимоотношения, или межличностный;                
      авторитарный;                                                         
      демократический;                                                      
      структурирующий (директивный);                                        
      коллаборативный, или основанный на сотрудничестве с работником (Cann, 
 Siegfried, 1990; Eagly, Johnson, 1990).                                    
 
      В данной методике учитывается пол менеджера и проявление в его  пове- 
 дении: либо гендерно типичных качеств (маскулинных - для мужчин,  феминин- 
 ных - для женщин), либо гендерно атипичных (маскулинных - для женщин,  фе- 
 мининных - для мужчин), либо гендерно нейтральных (характерных  для  обоих 
 полов).                                                                    
      Методика используется в 2 вариантах:                                  
      а) самооценочном (испытуемые-менеджеры сами оценивают свой стиль);    
      б) экспертном - по оценкам экспертов, которые хорошо знают  менеджера 
 по работе, деятельности (подчиненные, начальники, коллеги).                
      Можно использовать оба варианта по отношению к одним и тем же испыту- 
 емым.                                                                      
      Используются 2 набора качеств. Первый взят из широко известного  воп- 
 росника университета Огайо (LBDQ) - описание поведения двух лидерских сти- 
 лей (по 10 поведенческих проявлений на каждый стиль): а) доброжелательного 
 (центрированного на работнике) и б) структурирующего.  Второй  содержит  3 
 набора качеств: а) маскулинных; б) фемининных; в) гендерно-нейтральных.    
 
      Математическая обработка результатов осуществляется с помощью  ключа. 
 При этом по каждому из следующих показателей выводится  среднее  арифмети- 
 ческое:                                                                    
      1) маскулинность,                                                     
      2) феминность,                                                        
      3) гендерная нейтральность,                                           
      4) доброжелательный лидерский стиль (центрированный на работнике);    
      5) структурирующий лидерский стиль (центрированный на задаче).        
 
      Показатели считаются высокими, если среднее значение  составляет  4-5 
 баллов, средними, если среднее значение равно 3 баллам,  и  низкими,  если 
 среднее значение составляет 1-2 балла.                                     
      Сравнение результатов по различным показателям позволяет сделать  вы- 
 вод о лидерском стиле испытуемого.                                         
 
      Примечание к компьютерной версии.                                     
      В данной программе диаграмма, а  также  автоматическая  интерпретация 
 результатов тестирования основана на показателях выраженных в процентах  c 
 нулевой точкой отсчета.                                                    
      Условные критерии автоматической интерпретации:                       
       0% -  20%  низкий тестовый показатель;                               
      21% -  40%  пониженный тестовый показатель;                           
      41% -  60%  средний тестовый показатель;                              
      61% -  80%  повышенный тестовый показатель;                           
      81% - 100%  высокий тестовый показатель.                              
 
      Опросник состоит из 56 вопросов.                                      
      Примерное время тестирования 10-15 минут.

     1. Маскулинные качества              - Мс = 48  4.00  75%              
     2. Фемининные качества               - Фм = 22  1.83  21%

     4. Доброжелательный лидерский стиль  - ДС = 27  2.70  42%

                           ИНТЕРПРЕТАЦИЯ:                                   
 
      1. Маскулинные качества: жесткий, честолюбивый, авторитарный,         
 суровый, строгий, ассертивный, независимый,  конкурирующий,  доми-         
 нантный, агрессивный, уверенный в себе, аналитический, деятельный.         
      Тестовый показатель: повышенный.                                      
 
      2. Фемининные качества: восприимчивый  (благодарный),  легко-         
 возбуднмый, подчиненный, мягкий,  избирательный,  сентиментальный,         
 зависимый, нежный (любящий), сочувствующий, способный к пониманию,         
 эмоциональный, кроткий (мягкий).                                           
      Тестовый показатель: пониженный.                                      
 
      3. Гендерно-нейтральные качества: тактичный, искренний,  сис-         
 тематичный, квалифицированный, сознательный, официальный (торжест-         
 венный), оказывающий помощь, адаптивный, предсказуемый, правдивый,         
 традиционный, надежный.                                                    
      Тестовый показатель: средний.                                         
 
      4. Доброжелательный лидерский стиль: дружелюбен  и  доступен;         
 заботится о личном самочувствии членов группы; хочет внести  пред-         
 ложения; держит себя в руках; заранее предупреждает об изменениях;         
 объясняет свои действия; поддерживает предложения, сделанные груп-         
 пой во время выполнения задачи; рассматривает всех  членов  группы         
 как равных; делает все, чтобы было приятно  находиться  в  группе;         
 действует, консультируясь с группой.                                       
      Тестовый показатель: средний.                                         
 
      Структурирующий лидерский стиль: распределяет рабочие задания         
 между членами группы; сообщает группе,  что  ее  ожидает;  уверяет         
 членов группы в том, что лидер понимает групповые обязанности; со-         
 храняет определенные стандарты в достижении  результатов;  решает,         
 что должно быть сделано и как; раскрывает свои взгляды перед груп-         
 пой; поощряет применение единообразной процедуры; требует,  чтобы          
 члены группы придерживались стандартных правил; выносит на  группу         
 свои идеи; составляет график выполнения работы.                            
      Тестовый показатель: высокий.

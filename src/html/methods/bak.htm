МЕТОДИКА:                                  
                              Опросник "ВАК"                                
                 (Визуальная. Аудиальная. Кинестетическая.)                 
                             (В.А. Ананьев).                                
 
      Методика предназначена для диагностики репрезентативной системы  под- 
 ростка.                                                                    
      В последнее время значительное распространение получило  понятие  ре- 
 презентативных систем личности. Термин этот пришел из  нейро-лингвистичес- 
 кого программирования (НЛП) - одного из западных психотерапевтических  на- 
 правлений. Теория НЛП постулирует, что у каждого индивида существует пред- 
 почитаемая им репрезентативная система, которая называется ведущей  репре- 
 зентативной системой (ВРС). Люди используют репрезентативные  системы  для 
 построения внутренней модели мира, для переработки информации, для регуля- 
 ции поведения.                                                             
 
      Примечание к компьютерной версии.                                     
      В данной компьютерной программе диаграмма и интерпретация результатов 
 тестирования основаны на следующих условных критериях выраженных в процен- 
 тах:                                                                       
       0% -  20%  низкий показатель;                                        
      21% -  40%  пониженный показатель;                                    
      41% -  60%  средний показатель;                                       
      61% -  80%  повышенный показатель;                                    
      81% - 100%  высокий показатель.                                       
 
      Опросник состоит из 13 вопросов.                                      
      Примерное время тестирования 5-10 минут.

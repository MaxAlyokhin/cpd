МЕТОДИКА:                                  
                Шкала показателей групповых взаимоотношений                 
                        (В.Л. Марищук, Л.К. Серова).                        
 
      Методика предназначена для оценки взаимоотношений в группе путем  на- 
 блюдения. Наблюдение целесообразно осуществлять  по  следующим  критериям: 
 характер взаимодействия, уровень взаимопонимания, наличие одобрений и  уп- 
 реков, готовность к взаимопомощи.                                          
      Наблюдаемые признаки оцениваются по 5-балльной шкале.                 
 
      Методика состоит из 5 пунктов оценки.                                 
      Примерное время оценивания 5-10 минут.

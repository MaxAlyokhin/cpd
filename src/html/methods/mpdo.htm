МЕТОДИКА:                                  
      Модифицированный патохарактерологический диагностический опросник     
                                  (МПДО)                                    
                             (Ю.А. Парфенов).                               
 
      Методика предназначена для  идентификации типов акцентуаций характера 
 у подростков.                                                              
      Модификация ПДО:                                                      
      1. В текст опросника включены  только  диагностические  вопросы,  что 
 позволило уменьшить его объем с 351 до 143 вопросов.                       
      2. Диагностически значимыми считаются только ответы "Да", что  позво- 
 ляет провести обследование в один прием (а не в два, как в ПДО).           
      3. Значительная доля обработки результатов теста может проводится са- 
 мими испытуемыми: подсчет баллов на бланке ответов и выделение шкал с наи- 
 большей суммой баллов.                                                     
      4. Модификация коснулась и содержания опросника. Так, некоторые  воп- 
 росы получены путем анализа большого количества проекций  акцентуированных 
 подростков с помощью методики незаконченных предложений.                   
 
      Содержательная валидность МПДО проверялась двумя способами:           
      1. По корреляции диагнозов, полученных по МПДО, с  диагнозами,  полу- 
 ченными по ПДО. Совпадения в ответах по всем типам акцентуаций в  совокуп- 
 ности составили 76%.                                                       
      2. Проверялась корреляция полученных по МПДО диагнозов с  экспертными 
 оценками классных руководителей. Совпадение диагнозов, полученных по МПДО, 
 с диагнозами, выставленными экспертами на  основании  "школьной  клиники", 
 составило 87%.                                                             
 
      Надежность опросника проверялась методом повторного тестирования  че- 
 рез две недели. Подтвердилось 94% диагнозов.                               
 
      МПДО апробирован на 316 акцентуированных подростках 8-11 классов.     
 
      Опросник включает 143 утверждения, составляющих 10 диагностических  и 
 одну контрольную шкалу (шкалу лжи). В каждой шкале по 13 утверждений.  Ут- 
 верждения в тексте вопросника предъявляются в случайном порядке.           
      Диагностируются следующие типы акцентуаций:                           
      гипертимный,                                                          
      циклоидный,                                                           
      лабильный,                                                            
      астено-невротический,                                                 
      сенситивный,                                                          
      тревожно-педантический,                                               
      интровертированный,                                                   
      возбудимый,                                                           
      демонстративный,                                                      
      неустойчивый.                                                         
 
      На основании собранного материала, отдельно по каждому типу акцентуа- 
 ций, определено минимальное диагностическое  число  (МДЧ),  представляющее 
 собой нижнюю границу доверительного интервала, которая  рассчитывается  по 
 формуле:                                                                   
      МДЧ=М-Rn*W                                                            
 где                                                                        
      М - средний по выборке балл данного типа акцентуаций;                 
      Rn-табличный коэффициент;                                             
      W- размах данных.                                                     
 
      Контрольная шкала: показатель в 4 балла и  выше  рассматривается  как 
 критический. Высокий показатель по этой шкале свидетельствует о  тенденции 
 обследуемых давать "хорошие" ответы.                                       
 
      Правила идентификации типов:                                          
      1. Если МДЧ достигнуто или превышено только по одному типу, то  диаг- 
 ностируется этот тип.                                                      
      2. Если МДЧ превышено по нескольким типам, то диагностируется:        
      а) в случае совместимых типов - смешанный тип;                        
      б) если по какому-либо типу набрано баллов на 4 больше, чем по другим 
 типам, то последние не диагностируются, даже если сочетание совместимо;    
      в) в случае несовместимых сочетаний диагностируется тот тип, в пользу 
 которого набрано больше баллов;                                            
      г) если в отношении двух несовместимых типов имеется одинаковое коли- 
 чество баллов, то для исключения одного из них руководствуются т.н. " пра- 
 вилами доминирования";                                                     
      3. Если МДЧ достигнуто или превышено в отношении нескольких  типов  и 
 по правилу 2 их не удается сократить до двух, то тогда отбираются два типа 
 с наибольшим числом баллов, после чего руководствуются правилом 2.         
      4. Если по контрольной шкале получено более 4 баллов, то, к шкале де- 
 монстративности прибавляется 1 балл, если больше 7 - прибавляется соответ- 
 ственно 2 балла. Однако, если после этого демонстративный тип не  диагнос- 
 тируется, результаты обследования признаются недостоверными и в  отношении 
 данного подростка его надо повторить.                                      
      Достижение или превышение МДЧ по тому или иному типу  плюс  представ- 
 ленные выше правила идентификации типов гарантируются точность диагностики 
 не менее Р > 0,95.                                                         
 
      Примечания к компьютерной версии:                                     
      1. Некоторые правила определения типов акцентации  м. МПДО  отличатся 
 от тех, которые предложены в м. ПДО. В данной программе реализованы прави- 
 ла модифицированного варианта теста (м. МПДО).                             
      2. При определенных значениях тестовых  показателей  (например,  если 
 большинство шкал превышают диагностический порог, но,  либо  незначительно 
 отличатся между собой, либо равны между собой) количество  выраженных  ак- 
 центуаций может превышать 2. В этом случае с помощью программной обработки 
 результатов тестирования невозможно установить чистый или  смешанный  (со- 
 стоящий из 2 акцентуаций) тип. Подобные варианты программа определяет  как 
 "Мозаичный тип акцентуации". В данной ситуации  психолог  должен  уточнить 
 результаты тестирования с помощью привлечения дополнительной информации об 
 испытуемом или повторного тестирования. Для написания текстового  заключе- 
 ния по результатам тестирования можно  использовать  файл  альтернативного 
 варианта последнего отчета предназначенный для подобных случаев. Этот  ва- 
 риант отличается от основного последовательным описанием всех акцентуациий 
 личности которые были возможны при последнем  тестировании  даже  если  их 
 число превышает 2. После окончания тестирования выберите опцию  "Последний 
 отчет" Windows-оболочки программы. По умолчанию  откроется  основной  файл 
 последнего отчета. Далее выберите "Открыть" и в папке  "MPDO"  Вы  увидите 
 два текстовых файла - "mpdo.txt" и "mpdo_k.txt". Откройте последний и вне- 
 сите необходимые изменения.                                                
      3. В соответствии с правилом 4 определения типа, если по  контрольной 
 шкале получено более 4 баллов и диагносцируется демонстративный тип, авто- 
 матическая интерпретация контрольной шкалы в данной программе не  произво- 
 дится.                                                                     
 
      Опросник состоит из 143 утверждений.                                  
      Примерное время тестирования 30-35 минут.

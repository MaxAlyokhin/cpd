МЕТОДИКА:                                  
                    Опросник "Сенсорные предпочтения"                       
                    (Эйдемиллер Э.Г., Юстицкис В.В.).                       
 
      Методика предназначена для диагностики сенсорных предпочтений личнос- 
 ти.                                                                        
      Разработка опросника основывалась на том, что визуальные,  аудиальные 
 и кинестетические ощущения являются отдельными источниками  сенсорной  ин- 
 формации, т.е. высказывания опросника рассматриваются как внешние  индика- 
 торы различных ощущений, посредством которых испытуемые  воспринимают  ин- 
 формацию.                                                                  
      Опросник представляет собой набор восьми жизненных ситуаций,  поведе- 
 ние в которых может определяться предпочтением одной из четырех  сенсорных  
 модальностей. Первые четыре ситуации сконструированы по преобладающему ис- 
 точнику сенсорной информации, в частности для уравновешивания  вероятности 
 ответов различных модальностей. Метод парных сравнений  позволяет  "зашум- 
 лять" выбор той или иной репрезентативной системы. Многократно совершаемый 
 выбор в ходе заполнения опросника позволяет испытуемым четче осознать свои 
 сенсорные предпочтения.                                                    
      Заполнение испытуемым теста дает последовательность исходов,  которая 
 затем подвергается однозначной количественной интерпретации.               
      Суммарный балл подсчитывается с помощью ключа.                        
      Ключ устанавливает числовое значение исхода по каждому пункту. В кон- 
 це каждого опросника подсчитывается общее количество баллов испытуемого по 
 каждой сенсорной модальности. Каждая модальность встречается  в  тесте  24 
 раза, следовательно, максимальная оценка по каждой шкале равняется 24.     
 
      Опросник состоит из 48 вопросов.                                      
      Примерное время тестирования 10-15 минут.                             
 
      Примечание к компьютерной версии.                                     
      В данной компьютерной программе диаграмма и интерпретация результатов 
 тестирования основаны на следующих условных критериях выраженных в процен- 
 тах:                                                                       
       0% -  20%  низкий показатель;                                        
      21% -  40%  пониженный показатель;                                    
      41% -  60%  средний показатель;                                       
      61% -  80%  повышенный показатель;                                    
      81% - 100%  высокий показатель.

МЕТОДИКА:                                  
                 Шкала оценки условий эффективной тренировки                
                              (Б. Дж. Кретти).                              
 
      Методика предназначена для диагностики стимулов, условий,  помогающих 
 спортсмену эффективно тренироваться и выступать на соревнованиях.          
      Спортсмену предлагается оценить  по  десятибалльной  шкале  различные 
 факторы, помогающие ему хорошо тренироваться.                              
      После того как будет проведен опрос членов команды, полученную инфор- 
 мацию можно использовать следующим образом:                                
      1. Вычисляется сравнительная значимость каждого фактора. При  необхо- 
 димости в тренировочный процесс вносятся соответствующие коррективы.       
      2. Прежде чем анализировать средние показатели  ответов  спортсменов, 
 тренер может сам попробовать предсказать, как, по его мнению,  они  оценят 
 каждый из предлагаемых вопросов.                                           
      В результате сопоставления оценок тренера с действительными  оценками 
 можно получить представление о том, насколько мнение тренера  совпадает  с 
 установками большинства членов команды.                                    
 
      Опросник состоит из 9 вопросов.                                       
      Примерное время тестирования 3-5 минут.

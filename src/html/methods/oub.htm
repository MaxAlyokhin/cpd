МЕТОДИКА:                                  
                   Тест-опросник удовлетворенности браком                   
                 (В.В. Столин, Т.Л. Романова, Г.П. Бутенко).                
 
      Методика предназначена для экспресс-диагностики степени удовлетворен- 
 ности-неудовлетворенности браком, а также степени согласования-рассогласо- 
 вания удовлетворенности браком у той или иной социальной группы.           
      Опросник представляет собой одномерную шкалу, состоящую из  утвержде- 
 ний, относящихся к различным сферам: восприятия себя и  партнера,  мнения, 
 оценки, установки и т.д.                                                   
      Возможный диапазон тестового балла - от 0 до 48 баллов.               
      Высокий балл говорит об удовлетворенности браком.                     
 
                                  Нормы.                                    
      Различия между средними для разводящихся и  благополучных  достоверны 
 по критерию Стьюдента (t=10.835) на уровне значимости 0.01.                
      Доверительные интервалы с коэффициентом 0.95  для  средних  суммарных 
 баллов равны:                                                              
      для разводящихся (20,76; 23,26),                                      
      для "благополучных" (30,92; 33,34).                                   
      По критерию Фишера можно считать равными дисперсии  обеих  выборок  с 
 уровнем значимсоти 0.05.                                                   
      Для практического использования норм теста удобно взять целочисленные 
 значения средних и дисперсии из их доверительных  интервалов.  Будем  счи- 
 тать, что  x(н)=32, а x(р)=22, при этом их средние квадратичные равны 8.   
      Тогда, учитывая нормальность полученных распределений, можно  постро- 
 ить удобную таблицу для проведения экспресс-диагностики испытуемого по его 
 сумарному баллу. Вся ось суммарных баллов разбивается на 7 категорий:      
       0 - 16 баллов - абсолютно неблагополучные,                           
      17 - 22 баллов - неблагополучные,                                     
      23 - 26 баллов - скорее неблагополучные,                              
      27 - 28 баллов - переходные,                                          
      29 - 32 баллов - скорее благополучные,                                
      33 - 38 баллов - благополучные,                                       
      39 - 48 баллов - абсолютно благополучные.                             
 
      Опросник состоит из 24 вопросов.                                      
      Примерное время исследования 15-20 минут.
